# deepfake-video-detection
Final year degree project – Deepfake Video Detection using CNN with Tkinter GUI

## Overview
This project is my final year degree project focused on detecting deepfake videos
using deep learning techniques. The system uses a CNN architecture
to analyze spatial and temporal features from videos.

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- CNN
- Tkinter GUI

## Project Structure
- src/      : Source code
- models/   : Trained deep learning models (.h5)
- images/   : Accuracy and result screenshots

## Working
User uploads a video through the Tkinter GUI
Video frames are extracted using OpenCV
Each frame is processed and passed to the CNN model
The model analyzes spatial features
Final prediction is displayed as Real or Fake

## Note
Datasets and training videos are not uploaded due to large size.
